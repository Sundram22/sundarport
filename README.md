## ⚡ Quick Start

1. Folder download karo
2. `index.html` browser mein kholo
3. Done! 🎉

---

## 📊 Google Sheets Setup

1. `python-backend/google_apps_script.js` ka code copy karo
2. Google Sheet → Extensions → Apps Script → Paste → Save
3. `testDoPost` run karo → Allow karo
4. Deploy → New Deployment → Web App → Anyone
5. URL copy karo → `js/main.js` mein `GOOGLE_SCRIPT_URL` mein daalo

---

## 🌍 Vercel Deploy

```bash
git init
git add .
git commit -m "Portfolio V2"
git branch -M main
git remote add origin https://github.com/USERNAME/sundram-portfolio.git
git push -u origin main
```
Phir vercel.com pe import karo → Done!

## ✏️ Customize

- **WhatsApp number**: `index.html` mein `wa.me/919999999999` change karo
- **Resume**: `resume.pdf` naam ki file rakho folder mein
- **Email**: `sundram@example.com` replace karo
- **Colors**: `css/style.css` mein `:root` variables change karo
- **Typing texts**: `js/main.js` mein `typingTexts` array update karo
